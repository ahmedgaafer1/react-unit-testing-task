# Server (API)

This directory contains the backend server for the Cat Adoption App.

## Requirements

- Node.js

## Getting Started

1. Install dependencies:
   ```
   npm install
   ```
2. Start the server:
   ```
   npm start
   ```
3. The server will run by default at http://localhost:4000
